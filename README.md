# customer-segmentation-project



A complete customer segmentation project that groups customers based on behavioral and demographic data using machine learning clustering techniques. This project demonstrates how businesses can identify distinct customer groups, analyze purchasing patterns, and develop targeted marketing strategies.

## Project Overview

Customer segmentation is a key analytics technique used to divide customers into meaningful groups based on shared characteristics. In this project, K-Means clustering is used to identify customer segments using features such as age, income, and spending behavior.

### Objectives

- Segment customers using machine learning clustering algorithms
- Analyze customer purchasing patterns and preferences
- Visualize customer groups and their characteristics
- Generate actionable business insights from customer data

## Key Features

- Customer data generation for testing and development
- Data preprocessing and feature scaling
- K-Means clustering implementation
- Segment profiling and analysis
- Customer segment visualization
- Modular and GitHub-ready project structure


## Technologies Used

- Python 3.x
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Dataset Features

The generated dataset contains:

| Feature | Description |
|-----------|-------------|
| Age | Customer age |
| Income | Annual income |
| SpendingScore | Spending behavior score |
| Segment | Cluster assignment generated by the model |
